SMTP Setup Guide

Email reminders need SMTP env vars on Render:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
SMTP_SECURE=false

For Gmail: enable 2FA → create App Password. For other providers, adjust host/port.
